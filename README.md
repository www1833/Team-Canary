# カナリア軍団 — 公式サイト

Vite + React + TypeScript で構築された草野球チーム「カナリア軍団」の公式サイトです。Tailwind CSS と shadcn/ui ベースのコンポーネントを利用し、GitHub Pages でのホスティングを想定した完全クライアントサイド SPA として動作します。

## カラーパレット
- イエロー: `#FFDF00`
- グリーン: `#009739`
- ダークグリーン: `#0B3D2E`

## ロゴ
[lucide-react](https://lucide.dev/) の `Bird` アイコンと「カナリア軍団」のタイポグラフィを組み合わせたシンプルなロゴを利用しています。（後日差し替え可能）

## 主な機能
- Home: ヒーローセクション、featured ギャラリーのスライダー、チーム紹介、活動実績カウンター、参加案内 CTA。
- Members: 検索・ポジション/投打フィルタ・ソート・詳細モーダル付きのメンバーグリッド。
- Admin: ログイン後にメンバーとギャラリーの CRUD、ドラッグ＆ドロップ並び替え、JSON インポート/エクスポート、画像アップロード（自動縮小・サムネ生成）、パスワード変更を実行。
- データはブラウザの `localStorage` に保存され、初回アクセス時にはサンプルデータを投入します。

## 開発
```bash
npm install
npm run dev
```

## ビルド
```bash
npm run build
```

## デプロイ手順
1. GitHub にリポジトリを作成。
2. `vite.config.ts` の `base` をリポジトリ名に合わせて更新。
3. `npm run build`
4. `npm run deploy`
5. GitHub Pages で `gh-pages` ブランチを公開。

## 管理ログイン
- 初期 ID / パスワード: `admin` / `change-me`
- 管理画面の Security タブからパスワードを変更できます。

## 管理機能
### メンバー管理
- 追加 / 編集 / 削除
- ドラッグ＆ドロップによる並び替え
- JSON のインポート / エクスポート
- 画像アップロード時に 800px へ自動縮小

### ギャラリー管理
- 複数枚アップロード（EXIF 補正、最大 2048px へ縮小、400px サムネイル生成）
- featured フラグで Home のトップスライダーへ反映
- 検索 / タグフィルタ
- JSON のインポート / エクスポート

## 既知の制約
- 静的サイトのため認証情報は厳密には守れません。共有端末では利用後にログアウトし、定期的にパスワードを更新してください。
- `localStorage` の容量には限りがあるため、定期的に JSON をエクスポートしてバックアップを行ってください。
